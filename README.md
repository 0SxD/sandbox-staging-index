# sandbox-staging-index

Pointer index for SandBoxSetup. Each area has a NOTES.md with good/bad annotations and source paths. NO PII content is uploaded to this repo -- only path references and orchestrator commentary.

Created 2026-04-30 by CLI sweep session for Claude Desktop review.

## How to use

Each subfolder mirrors a SandBoxSetup top-level directory or area. Open the NOTES.md in each subfolder to see:
- What lives at the source path
- Good (publishable / supports application narrative)
- Bad / risk (PII, codenames, evaluation platform terms, etc.)
- Desktop action (read / skip / verify)

Source paths are relative to `<staging_path>\` on Sage's local machine.
